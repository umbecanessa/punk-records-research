# E10.1 — GPT-like objective (open phrasing + live chat)

**Status:** in progress — train `encoder_e10a_best.pt` then validate  
**North star:** talk naturally, remember in STORAGE, save tokens vs chat replay

---

## What changed from E10 v1

| v1 | v1.1 (GPT path) |
|----|-----------------|
| 12-char NL window | **96-char** utterance window |
| Template phrasing | **Open conversational** corpus |
| Batch line chat | **Live Gradio Chatbot** tab |
| E9a parser | **E10a NlParserModel** (standalone, ~2M params) |

---

## Architecture (unchanged law)

```
Natural chat → E10a parser (96-char) → PLANT | QUERY | CHITCHAT
                      │
         CHITCHAT → OBS only
         PLANT/QUERY → kernel STORAGE → E9b renderer → reply
```

---

## Train & validate

```bash
python -m greenfield.train.train_encoder_e10a --device cuda
python -m greenfield.eval_open_phrasing
python -m greenfield.eval_chat_v1
python -m greenfield.validate_release   # + open_phrasing gate
```
